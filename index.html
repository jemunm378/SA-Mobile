What is SA Mobile?

SA Mobile (Also know as Scratch Addons Mobile), is a fork of Scratch Addons (which is created by World_Languages and others). SA Mobile is aiming for scratch mobile users (IOS users)

Features

Various features to empower your Scratch website and editor experience

Easy-to-use interface with various controls for each feature and theme

Free, open source, and maintained by many users from the Scratch community

Chrome Firefox
